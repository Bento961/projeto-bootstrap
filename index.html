O que é Bootstrap?

Bootstrap é um framework front-end gratuito e de código aberto para desenvolvimento web responsivo e móvel.

Bootstrap facilita o desenvolvimento de páginas web com uma variedade de componentes e classes CSS prontas para uso. Documentação Bootstrap

Componentes Bootstrap

O Bootstrap oferece diversos componentes reutilizáveis, como botões, cards, e barras de navegação.

Os componentes do Bootstrap são projetados para serem altamente personalizáveis, permitindo que você crie interfaces atraentes e funcionais. Componentes Bootstrap

Grid System do Bootstrap

O sistema de grade do Bootstrap ajuda a criar layouts responsivos com facilidade.

Utilizando uma grade flexível baseada em colunas, você pode organizar e alinhar o conteúdo da sua página de forma eficaz. Sistema de Grid do Bootstrap
